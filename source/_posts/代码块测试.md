# 代码高亮测试

这是我的博客样式测试页面。主要用于验证 **Hexo-Fluid** 主题的 `highlight` 配置是否生效。

## 1. Java 代码块测试
测试目标：验证 Java 关键字、注解、字符串和注释的配色。

```java
package com.zhushuohaiya.blog;

import java.util.List;

/**
 * 这是一个测试类
 * 用于验证代码高亮效果
 */
public class HighLightTest {

    // 主函数入口
    public static void main(String[] args) {
        System.out.println("Hello, Fluid Theme!");
        
        // 测试循环结构
        for (int i = 0; i < 5; i++) {
            printStatus(i);
        }
    }

    private static void printStatus(int index) {
        // 测试数字高亮
        if (index % 2 == 0) {
            System.out.println("Even Number: " + index);
        } else {
            System.err.println("Odd Number: " + index);
        }
    }
}
